새로운 여행지 추가 절차 🚀

1단계: 여행 계획을 엑셀이나 텍스트로 자유롭게 정리합니다.
2단계: 제미나이(AI)에게 엑셀 파일을 주며 "우리 앱용 JSON 코드로 변환해줘" 라고 요청합니다.
3단계: 구글 스프레드시트의 Library 탭을 엽니다.
4단계: A열에 조건(예: 방콕_5일)을 적고, B열에 제미나이가 준 JSON 코드를 복사해서 붙여넣습니다.
5단계: 앱의 설정 탭에서 A열에 적은 조건과 똑같이 맞춘 뒤 '일정 생성'을 누르면 즉시 화면에 나타납니다!
